calcClaim2 engine & PDF test alignment

The calcClaim vs calcClaim2 comparison document describes a modular claim-calculation pipeline and a per-component testing strategy (focused unit tests instead of one 12k-line integration test). This demo runs that pipeline on the claims path; the table below maps each C++-named module to pytest coverage.
